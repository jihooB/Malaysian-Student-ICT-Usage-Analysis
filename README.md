Analysis of Primary Students' ICT Usage for Learning in Malaysia

This project performs an econometric analysis of the factors influencing computer usage among primary school students in Malaysia for educational purposes. 
By utilizing Logit and Probit models, the study identifies how demographic characteristics and UTAUT (Unified Theory of Acceptance and Use of Technology) variables impact students' ICT engagement.
